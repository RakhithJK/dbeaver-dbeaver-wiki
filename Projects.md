# Projects

## Filtering objects

Each database section has in Projects window (ALT + W + P) has an option to filter in or out specific objects. You can filter schemas, tables, views and procedures. The name of the section changes to include `(...)` marker if the filter is currently enabled.

To quickly filter out specific tables select them and use the option _Filter N objects_:

When defining the names of the objects to filter you can use SQL glob-like chars `%` and `_`.
